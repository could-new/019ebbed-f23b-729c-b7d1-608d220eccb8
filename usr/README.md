# Math Vibes Podcast App

Современное Flutter-приложение для прослушивания и чтения сценариев математического подкаста. Проект создан специально для детей (9-14 лет) и взрослых, где 12-летний ведущий рассказывает о математике через призму современных трендов, игр и поп-культуры.

## Особенности
- **Темная тема**: Стильный молодежный дизайн в темных тонах с неоновыми акцентами.
- **Список эпизодов**: Удобная навигация по темам подкаста.
- **Детальные сценарии**: Текстовые версии подкастов, написанные живым и эмоциональным языком с wow-эффектами.
- **Кроссплатформенность**: Работает на iOS, Android, Web и Desktop.

## Запуск
1. Убедитесь, что установлен Flutter.
2. Выполните \`flutter pub get\`.
3. Запустите приложение: \`flutter run\`.
